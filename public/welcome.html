// FEATURES

核心功能特性

打破传统终端限制，赋予 AI 智能体开发任务持久化、可视化与跨设备掌控力。

AI 智能体后台持续运行

一键启动 Agy、Claude Code 等 AI 编程助手。任务在服务器 Tmux 会话中独立执行，即使电脑休眠、断网或关闭网页也不中断，重连即可瞬间恢复现场。

多用户沙盒与安全隔离

完整的多用户管理模式。非管理员用户拥有独立的文件目录与虚拟运行环境，Shell 历史与个人配置完全隔离，保障开发数据的机密性与安全。

设备原生消息推送

当 AI 完成长耗时任务，或关键操作需要您授权时，系统会向手机或电脑发送原生推送通知 —— 即使没打开网页也能第一时间掌握状态。

微信与 Telegram 远程控制

扫码即可绑定微信或 Telegram。在聊天框中直接接收 AI 关键动作通知、查看终端实时截图、发送文本命令并一键审批，省去打开电脑的麻烦。

超低延迟交互终端

网页内提供响应敏捷的虚拟控制台，键盘反馈迅速。终端尺寸自动跟随浏览器窗口对齐，保障各种设备上的顺畅体验。

严密的安全认证

基于成熟的安全加密与 Token 校验，网页端采用管理员专属邀请码注册机制，全方位保护服务器命令行不被未授权访问。

// SHOWCASE

图形化工作台预览

多维度图形化看板，轻松掌控多智能体并行任务与代码编辑流程。

01 / 快速配置与初始化工作区

为不同的项目分支快速创建独立工作空间，随时导入现有代码库或进行空项目初始化，所有配置通过可视化列表完成。

  • Git 仓库对接：输入 Git 地址一键克隆并映射为专属工作区。
  • 安全沙盒路径：普通用户工作区无法越权访问服务器敏感目录。
  • 可视化工作区树：一目了然列出所有活跃目录，一键启动挂载会话。

02 / AI 智能体并行监控

支持创建多个运行中的 AI 智能体终端，侧边栏清晰列出挂载状态，可观察 AI 运作时的每一步分析与编码动作。

  • 常用智能体预置：内置 Agy、Claude Code 等环境，一键启动。
  • 多任务并发运行：多个独立 AI 会话各自处理不同开发需求。
  • 操作步骤拦截：关键动作自动暂停等待，并在微信/TG 推送审批。

03 / 文件修改直观对比 (Diff)

在控制面板中直接审阅 AI 对文件做出的所有代码修改，清晰的 Diff 差分让您无需运行 Git 命令即可完成合规检视。

  • 直观差异高亮：红绿底色清晰展示修改、增加及删除行。
  • 轻量级只读浏览器：快速审查庞大的文件修改，安全可靠。
  • 侧边栏文件树：双击检索工作区文件，多终端快捷联动。

04 / 完美适配手机移动设备

手机与平板浏览器界面完美自适应，支持作为独立桌面图标运行，出门在外也无需担心开发进度。

  • 触控虚拟键盘适配：防止输入框在移动设备上变形或溢出遮挡。
  • 添加至手机主屏幕：支持 PWA，像原生 App 一样全屏沉浸运行。
  • 息屏接收关键通知：阻断性审批即时推送至手机系统层面。
