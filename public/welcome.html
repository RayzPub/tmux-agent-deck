// FUNCTION OVERVIEW

核心功能特性

打破传统终端限制，赋予 AI 智能体开发任务持久化、可视化与跨设备掌控力。

AI 智能体后台持续运行

支持一键启动 Agy、Claude Code 等 AI 编程助手。任务在服务器的 Tmux 会话中独立执行。即使电脑休眠、断网或关闭网页，AI 也不会中断任务，再次连接即可瞬间恢复现场。

多用户沙盒与安全隔离

提供完整的多用户管理模式。非管理员用户拥有独立的文件存放目录与虚拟运行环境。各用户的 Shell 历史记录及个人配置完全隔离，保障开发数据的机密性与安全性。

设备原生消息推送

集成消息通知机制。当 AI 运行长耗时任务完毕，或在关键操作需要您授权同意时，系统会向您的手机或电脑发送原生通知消息，即使您此时没有打开网页也能及时获取状态。

微信与 Telegram 远程控制

无需繁琐配置，扫码即可绑定微信或 Telegram。您可以直接在聊天框接收 AI 的关键动作通知，查看终端实时截图，或直接发送文本命令和一键审批操作，省去打开电脑的麻烦。

超低延迟交互终端

在网页中提供响应敏捷的虚拟控制台，键盘打字反馈迅速。终端大小会自动根据您的浏览器窗口尺寸进行对齐，保障各种设备上的顺畅使用体验。

严密的安全认证

基于成熟的安全加密和 Token 校验。网页端支持管理员专属的邀请码注册机制，全方位保护您的服务器命令行不会被未经授权的外部用户所接触。

// INTERFACE SHOWCASE

图形化工作台预览

多维度图形化看板，轻松掌控多智能体并行任务与代码编辑流程。

01 / 快速配置与初始化工作区

为不同的项目分支快速创建独立的工作空间。您可以随时导入现有的代码库或进行空项目初始化，所有的配置都通过可视化列表完成。

  • Git 仓库对接：支持输入 Git 地址一键克隆并映射为专属工作区。
  • 安全沙盒路径：普通用户的工作区无法越权访问服务器上的其他敏感目录。
  • 可视化工作区树：一目了然列出所有活跃目录，一键即可启动对应的挂载会话。

02 / AI 智能体并行监控

支持创建多个运行中的 AI 智能体终端，并在侧边栏列表中清晰列出挂载状态。您可以观察 AI 运作时的每一步分析与具体编码动作。

  • 常用智能体预置：内置 Agy、Claude Code 等环境，无需复杂的安装配置即可一键启动。
  • 多任务并发运行：支持开启多个独立的 AI 会话，各自处理不同的修复与开发需求。
  • 操作步骤拦截：AI 执行关键动作时会自动进入暂停等待状态，并在微信/TG中推送用户审批。

03 / 文件修改直观对比 (Diff)

在控制面板中直接审阅 AI 智能体对文件做出的所有代码修改。清晰的 Diff 差分让您无需运行 Git 命令就能完成代码合规性检视。

  • 直观差异高亮：用红、绿底色清晰展示文件的修改、增加及删除行。
  • 轻量级只读浏览器：方便快速审查庞大的文件修改，确保 AI 编码安全可靠。
  • 侧边栏文件树：随时双击检索工作区文件，实现多终端与文件的快捷联动。

04 / 完美适配手机移动设备

无论您使用的是手机还是平板浏览器，界面均可完美适应。支持作为独立桌面图标运行，提供优化的移动端触控，让您出门在外也无需担心开发进度。

  • 触控虚拟键盘适配：经过专门优化，防止输入框在移动设备上发生变形或溢出遮挡。
  • 添加至手机主屏幕：支持 PWA 规范，可以像原生 App 一样全屏沉浸运行。
  • 息屏接收关键通知：即使手机屏幕关闭，AI 产生的阻断性审批也会即时推送至手机系统层面。
